# Personal Portfolio - Maulidi Iddi Kitereja

This is a modern, responsive personal portfolio website designed for **Maulidi Iddi Kitereja**.

## Features
- **Modern Dark Theme**: Designed with a sleek, professional dark aesthetics using glassmorphism effects.
- **Responsive Design**: Fully responsive layout that looks great on mobile, tablet, and desktop devices.
- **Interactive Elements**: Includes smooth scrolling, scroll animations, and a mobile navigation menu.
- **Sections**:
  - **Hero**: Introduction with a call to action.
  - **About**: Brief biography and stats.
  - **Skills**: Highlight of technical expertise.
  - **Projects**: Showcase of past work (Stock System, Network Setup, etc.).
  - **Contact**: Contact form and social media links.

## How to Use
1.  **Open**: Simply double-click the `index.html` file to open it in your web browser.
2.  **Customize**:
    -   **Text**: Open `index.html` in a text editor (like VS Code or Notepad) to update the text in the About, Skills, and Projects sections.
    -   **Images**: You can replace the placeholder images or adjust the CSS in `styles.css`.
    -   **Social Links**: Update the `href="#"` attributes in the Contact section of `index.html` with your actual social media profiles.

## Technologies Used
-   **HTML5**
-   **CSS3** (with Flexbox, Grid, and Animations)
-   **JavaScript** (for interactivity)

## Files
-   `index.html`: The main structure of the website.
-   `styles.css`: All the styling rules.
-   `script.js`: JavaScript for animations and mobile menu.
